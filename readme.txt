Hello! Welcome to most profitable trading strategies searcher.

 Here you will get data about 40 different strategies profits or losses.
 Based on this data you will be able to decide which strategy to choose.
 All you need is to provide cryptocurrency pair and period you are
 interested in.

 Steps which are performed by program are:
 1) download real historical data from Binance.eu exchange for provided period.
 2) run simulation on provided period and cryptocurrency pair, generate HTML for each strategy.
 3) scrape all HTML files of simula tions into .txt file.
 4) show all data in a html page.

 Be patient. Steps 1 and 2 takes some time. But don't worry all process
 will be shown on a screen while you are doing anything else.
